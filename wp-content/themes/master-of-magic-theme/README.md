# Master of Magic theme

Master of Magic theme is a custom WordPress block theme designed for Master of Magic's website.

## Features

- Block-based layout for easy customization
- Optimized for performance and speed
- Responsive design for all devices
- Accessibility-ready

## Development

To start with development, clone the repository and set up a local WordPress environment. Make sure to have the latest version of WordPress installed.

1. Navigate to the theme directory:
   ```
   cd master-of-magic-theme
   ```

2. Install the required dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm start
   ```
